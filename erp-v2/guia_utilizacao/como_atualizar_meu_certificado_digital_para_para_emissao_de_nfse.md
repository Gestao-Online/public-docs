# Como Adicionar/Atualizar meu certificado digital para emitir NFSe?

Dentro do Gestão Online, caso tenha adquirido o módulo de emissão de NFSe, será necessário adicionar seu certificado digital para que o sistema tenha autorização de emitir as notas fiscais. Esse procedimento é feito dentro da tela [Empresas](/erp-v2/funcionalidades/parametrizacoes/empresas.md).

{% hint style="warning" %}
**Atenção:** Este mesmo procedimento é necessário caso o certificado digital tenha sido renovado, sendo necessário anexar novamente na tela Empresas para continuar emitindo notas no sistema.
{% endhint %}

![](/erp-v2/assets/guia_utilizacao/como_atualizar_certificado_01.gif)

Com a tela Empresas aberta, logo abaixo no espaço reservado para **Dados Fiscais**, encontrará o campo Titular do Certificado. Clique no botão <img src="/erp-v2/assets/icon_dropdow.png" alt="" data-size="line"> para ter acesso à opção <img src="/erp-v2/assets/btn_anexar_certificado.png" alt="Anexar certificado" data-size="line">. Ao clicar nesta opção, uma nova janela será exibida.

![](/erp-v2/assets/funcionalidades/parametrizacao/aba_empresas_add_campo_certificado.png)

{% hint style="warning" %}
**Importante:** Os certificados digitais são aceitos em **`.pfx`** ou **`.p12`**.
{% endhint %}

![](/erp-v2/assets/funcionalidades/parametrizacao/aba_empresas_add_certificado.png)

Após anexar o certificado, digite a senha e clique no botão `Validar e Salvar`. Agora, basta salvar as alterações feitas na tela Empresas, e estará pronto para emitir notas fiscais novamente no Gestão Online.

![](/erp-v2/assets/guia_utilizacao/como_atualizar_certificado_02.png)
